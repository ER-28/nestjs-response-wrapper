## Prerequisites

- Node.js >= 18
- NestJS >= 10

## Install

```bash
# npm
npm install nestjs-response-wrapper

# pnpm
pnpm add nestjs-response-wrapper

# yarn
yarn add nestjs-response-wrapper
```

The library has `@nestjs/common` and `@nestjs/core` as peer dependencies — they should already be installed in your NestJS project.
